# URLでんこーだー
URLをエンコードしたりデコードしたりするツール

## 使い方
１：変換タイプを選択。

２：「変換前」に変換するURLを入力。

３：「変換」ボタン押下。

４：「変換後」に変換されたURLが出力される。

５：「コピー」ボタン押下で変換後URLがコピーされる。
